Front + Back End Development | Web Design
Free Association
is a newly established workshop and publishing platform
working to promote engaging writing within the arts. Working closely with designer and client
throughout ideation I built the website, connecting it to a WordPress backend. See more

Having surveyed the options, I decided on WordPress for its great flexibility, low ongoing costs as well as being well paired with the client’s previous knowledge of CMS interfaces. This gave me the opportunity to learn basic PHP, whilst further developing my front-end coding skills.

The resulting backend was an intuitive experience for the client - using tags and custom fields to organise the content. The website itself was built largely as a single page application - providing fluid navigation for users.

Front + Back End Development | Web Design
Website portfolio design and build for Chicago based architect James Carter ⇗
. We collaborated closely working
towards an immersive design which voiced the engaging, playful
nature of his practice. See more
Crucial to James’ vision for the website was a desire to be presented with equal weight as both an academic and practicing architect. This lead me to a design with a flat hierarchy. Wanting to provide an immersive, smooth user experience I hard coded the site — eliminating Page loads by loading content dynamically. In order to achieve this careful detail had to be taken throughout the build, ensuring the website was lightweight enough to work as a single page application. The resulting website is one which is at once challenging and rewarding — standing out from the norm to provide a distinct voice to James’ practice.
